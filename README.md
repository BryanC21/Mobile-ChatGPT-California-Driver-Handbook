# Mobile-ChatGPT-California-Driver-Handbook
Mobile interface for interacting with fine tuned GPT api.   
API that uses: 
* GPT-3
* OpenAI API
* LlamaIndex
* LangChain

to answer questions about driving in California using knowledge from the 2023 driver's manual.
